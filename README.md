# medical-checkup-analysis
# Анализ прохождения медицинских осмотров студентов

## Описание проекта

Проект посвящен анализу процесса прохождения медицинских осмотров студентами.

Цель проекта:
- контроль прохождения медосмотров;
- выявление групп с низкой явкой;
- анализ нагрузки по датам;
- мониторинг статусов студентов.

## Инструменты

- Python
- Pandas
- Matplotlib
- Excel

## Статус проекта

В разработке.
